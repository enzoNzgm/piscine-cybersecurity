# Ex02 - Scorpion

Description
- Le programme Scorpion reçoit des fichiers image en paramètres et doit être capable de
les analyser pour en extraire les métadonnées EXIF et autres, puis de les afficher à l'écran

Installation
- Aucun build attendu si l'exécutable est fourni. Assurez-vous des permissions :

```bash
chmod +x scorpion
```

Contenu
- `scorpion` : exécutable fourni

Usage
- Obtenir l'aide du binaire (si disponible) :

```bash
./scorpion FILE1 [FILE2 ...]
```
